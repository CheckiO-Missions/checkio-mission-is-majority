How to count True-s in the list?

Use list method list.count() with argument True on items. 

return items.count(True) ...

Help me to check if counted amount is the majority of the list elements.

Majority means more than a half. Use built-in len() function on items and find a half from the result (divide into 2). It should be strictly less then the amount of True-s. 

return items.count(True) > len(items) / 2

Is this the whole solution?

You just need to add the definition of your function.

Here is everything together: 

def is_majority(items: list[bool]) -> bool:
    
    return items.count(True) > len(items) / 2
