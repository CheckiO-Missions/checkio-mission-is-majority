Jak policzyć True-s na liście?

Proszę użyć metody listy list.count() z argumentem True na items. 

return items.count(True) ...

Proszę mi pomóc sprawdzić, czy zliczona kwota stanowi większość elementów listy.

Większość oznacza więcej niż połowę. Proszę użyć wbudowanej funkcji len() na items i znaleźć połowę z wyniku (podzielić na 2). Powinna ona być dokładnie mniejsza niż ilość True-s. 

return items.count(True)> len(items) / 2
