多数意味着超过一半。在items 上使用内置的len() 函数，从结果中找出一半（除以 2）。它应该严格小于True-s 的数量。 

return items.count(True)> len(items) / 2

你只需要添加你的函数的定义。

def is_majority(items: list[bool]) -> bool:
    
    return items.count(True)> len(items) / 2

由于Python将True 解释为1 和False 解释为0 ，你只需将 找出items 的元素之和，它应该是 应该严格大于长度的一半。 

def is_majority(items: list[bool]) -> bool:
    
    return sum(items)> len(items) / 2
